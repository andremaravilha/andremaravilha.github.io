# andremaravilha.github.io

This repository contains the source code of my personal website published in [GitHub](https://andremaravilha.github.io/).

## How to build the project

To build this project you will need [npm](https://www.npmjs.com/), [Bower](https://bower.io/) and [Gulp](https://gulpjs.com/). Inside the root directory of the project, run the following commands:
```
npm install --global gulp-cli
npm install --save-dev
npm run build
```

All dependencies will be downloaded and a `dist/` folder will be created with the file for the production site.
